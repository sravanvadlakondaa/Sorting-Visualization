Sorting Algorithms Visualization






Example Array: [2, 5, 4, 7, 5]

This array will be used to visualize the sorting algorithms.

Each algorithm will reorder the array from unsorted to sorted form.

Sorting Algorithm Explanation

Bubble Sort: Bubble Sort repeatedly steps through the list, compares adjacent elements, and swaps them if they are in the wrong order. This process is repeated until the list is sorted.

Insertion Sort: Insertion Sort builds the sorted array one element at a time. It takes one element at a time and places it in the correct position among the previously sorted elements.

Selection Sort: Selection Sort works by repeatedly finding the minimum element from the unsorted part of the list and swapping it with the first unsorted element.

Merge Sort: Merge Sort divides the array into two halves, sorts each half recursively, and then merges the two sorted halves together in order.

Quick Sort: Quick Sort picks a pivot element and partitions the array into two sub-arrays. The pivot is placed in its correct position, and the process is repeated for the left and right sub-arrays.

Heap Sort: Heap Sort builds a max-heap from the array and then repeatedly extracts the maximum element to sort the array.

Cocktail Sort: Cocktail Sort is a variation of Bubble Sort. It moves in both directions across the array, sorting elements in each pass and shrinking the range of unsorted elements after each pass.

Gnome Sort: Gnome Sort works by comparing an element with the previous element and moving it back if it's in the wrong order. If it's in the right order, the algorithm moves forward.
